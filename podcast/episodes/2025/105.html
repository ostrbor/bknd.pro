Многоагентные системы и Copilot Studio Появился режим multi‑agent orchestration: Copilot Studio теперь позволяет строить системы из нескольких агентов, задачей которых является совместное выполнение сложных процессов

GitHub Copilot как активный агент Copilot развивается в peer‑программиста, способного назначать задачи, создавать pull‑requestы, исправлять баги и рефакторить код. Можно текстом дать задачу сделать PR на github'е и из github'а делегировать Copilot'у решение задачи.

NLWeb – разговорный web-интерфейс Появился Natural Language Web (NLWeb) — протокол для превращения сайтов в агентно‑доступные приложения, с которыми можно работать на естественном языке. Это готовый инструмент, который предоставляет /ask endpoint для того чтобы любой LLM-агент мог сделать запрос на ваш сайт и получить релевантный ответ. Под капотом NLWeb:

  • может работать с разными векторными базами данных;
  • для векторизация текста может использовать разные модели.
